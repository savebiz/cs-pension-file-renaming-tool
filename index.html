Professional File & Folder Management System

📁

Folder Renaming by PEN

Renames folders based on unique PEN numbers found in PDF files within each folder. Ensures each folder contains files with only one PEN number.

Example: SEPARATOR0001105911 → PEN200931411216 (Based on PEN200931411216_Document.pdf found inside)
📄

File Naming Convention

Standardizes PDF filenames by moving PEN numbers to the front, following the agreed naming convention.

Example: Document_PEN200931411216.pdf → PEN200931411216_Document.pdf
🗂️

PDF Separation by PEN

Sorts multiple PDFs in a single folder into separate subfolders based on their PEN numbers for better organization.

Example: /SEPARATOR0001100537/ ├── PEN100537001_Doc1.pdf → /PEN100537001/ ├── PEN100537002_Doc2.pdf → /PEN100537002/ └── PEN100537003_Doc3.pdf → /PEN100537003/
